# EaseAi

## To use audio from microphone you need to install these two libaries (win10)

### Get started

    1. pipenv install
    2. pipenv shell
    3. pip install pipwin
    4. pipwin install pyaudio

### List of commands

    - wikipedia     - open code
    - open youtube  - play music
    - open reddit   - the time
    - the day       - thank you
    - joke          - quit app
